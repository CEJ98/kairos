## 🛠️ Tech Stack
- Next.js 14 (App Router)
- Prisma + PostgreSQL
- Redis (Upstash)
- NextAuth.js
- TailwindCSS + shadcn/ui

## 🔒 Privacy First
- PostHog (self-hosted analytics)
- Sentry (error tracking)
- No third-party tracking
